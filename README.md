# 校园小秘(SchoolSecretary)
一款用于帮助教师高效管理学生的Android应用。<br>
<br>
本应用主要功能有抽签、点名、学生管理、成绩分析、教师交流（期待）等功能。<br>
Windows端在2015年重庆市电脑制作大赛中荣获市级一等奖，Android端和Web端正在开发中。<br>
<br>
开发者为重庆市巴蜀中学高2016级信息竞赛小组成员，现为一名高三学生，是Android开发初学者。<br>
